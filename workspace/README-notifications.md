## 🔧 How It Works:

The system uses your Tasker MCP server to bridge to Android notifications and SMS. When I respond:

1. Creative header is generated (random from 30+ verbs)
2. Toast notification appears on your device
3. Device vibrates briefly
4. SMS is sent with my actual message

When gateway goes down:
1. Emergency header with 🚨 emoji
2. Long vibration pattern
3. Emergency SMS alert
